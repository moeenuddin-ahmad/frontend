# TenantHub — Frontend

## ⚙️ Environment Variables

Create a `.env` file in the `frontend/` directory:

```env
VITE_API_URL=http://localhost:3000
```

## 🛠️ How to Run

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

App runs at **http://localhost:5173**

### Production Build

```bash
npm run build
```
